## 핵심 파일

### `agent/todo-logger.md`
서브에이전트 정의 파일. Claude Code에 복사해서 사용:
- 언어 감지 및 번역 로직
- 세션 ID 생성
- 이중 파일 작업 (sessions + by-date)
- 중복 감지 및 에러 처리

### `deploy-to-github.sh`
GitHub 배포 자동화 스크립트:
- Git 설정 확인
- GitHub CLI로 저장소 생성
- 자동 push 및 release 생성

### `examples/`
실제 로그 파일 예시로 포맷 이해 가능

### `docs/`
- **ARCHITECTURE.md**: 기술 문서 (엔지니어용)
- **INTEGRATION.md**: 상세 통합 가이드
- **DESIGN.md**: 한국어 설계 문서
- **간단한 docs**: 배포.md, 개요.md, 구조.md, 흐름.md (빠른 참조용)
