# 개요

AI 어시스턴트와 작업할 때 작업 목록은 현재 세션에만 존재합니다. 대화가 끝나면 히스토리가 사라집니다.

## 핵심 아이디어

Todo-Logger는 Claude Code의 모든 TodoWrite 작업을 자동으로 캡처해서 마크다운 파일로 저장합니다. Git 커밋 메시지 작성이나 일일 스탠드업에서 "어제 뭐 했더라?" 고민할 필요가 없어집니다.

## 작동 원리

1. Claude Code에서 TodoWrite 실행
2. todo-logger 서브에이전트 자동 호출
3. 작업 내용을 두 곳에 저장:
   - `sessions/YYYYMMDD-HHMMSS.md` - 시간순 세션 로그
   - `by-date/YYYY-MM-DD.md` - 날짜별 집계
4. 영어 작업은 자동으로 한국어로 번역

## 주요 특징

- **자동 영속화**: 수동 개입 없이 모든 작업 자동 기록
- **이중언어 지원**: 영어 ↔ 한국어 자동 번역
- **이중 구조**: 세션별 + 날짜별 뷰
- **Git 친화적**: grep으로 쉽게 검색 가능한 마크다운
- **실시간 로깅**: 2초 미만 처리
- **스마트 중복 제거**: 같은 세션 내 중복 방지

## 사용 사례

### Git 커밋 메시지
오늘 작업한 내용을 파일에서 바로 복사해서 커밋 메시지로 사용

### 일일 스탠드업
어제 완료한 작업(`✅`)과 오늘 계획(`🔄`, `🕐`)을 즉시 파악

### 프로젝트 회고
특정 기능 작업 히스토리를 grep으로 검색

### 팀 투명성
todo-history 폴더를 Git에 커밋해서 팀원들과 AI 작업 진행 상황 공유
